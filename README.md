# `rn-handy-utils`

Set of useful utils for React Native.

# Getting started

Install the library using either Yarn:

```
yarn add rn-handy-utils
```

or npm:

```
npm install rn-handy-utils
```

# Includes

- **Platform utils**:
  - isIos
  - isAndroid
  - isIosVersion
  - isAndroidVersion
- **Dimensions utils**:
  - getHeightFromRatio
  - getWidthFromRatio
  - getAspectRatio
- **Optimization utils**:
  - disableConsoleLogsInProd
- **HOOKS** :
  - useComponentSize
  - useTextLines
- **Other utils** :
  - openLink
  - mailto
